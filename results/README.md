 Frozen CAS outputs

These files preserve the main external computer-algebra outputs quoted in the manuscript. They are intended as lightweight reproducibility targets for readers who do not have Singular or Wolfram/Mathematica installed.

| File | Manuscript reference | Description |
| --- | --- | --- |
| `replication_minAssChar_22_components.txt` | Section 6.1 / Appendix 8.1 | The 22 irreducible components from Singular `minAssChar` for the self-replication ideal. |
| `two_pathway_elimination_minAssChar_components.txt` | Section 6.2 | The six parameter-only components obtained after eliminating `t1,...,t9` and applying `minAssChar`. |
| `two_pathway_facstd_first5_components.txt` | Appendix 8.2 | The first five components of the 92-component Singular `facstd` output for the two-pathway enzyme ideal. |

The scripts used to reproduce these outputs are in `singular/` and `mathematica/`.
